# 办公室绿萝 Pro

> 办公室绿萝生长监测系统 - Java + Vue 重构升级版

## 技术栈

- **前端**: Vue 3 + Element Plus
- **后端**: Java (Spring Boot)
- **数据库**: SQLite
- **存储**: 本地文件系统 + 图片
- **模型**: 本地AI模型 (YOLOv8 + ONNX)

## 项目状态

**当前版本**: v3.0.0 (规划中)

**开发阶段**: 基础设施搭建

## 产品定位

v2.2.0 (Python) → **办公室绿萝 Pro** (Java + Vue + AI模型)

- AI模型独立运行
- 叶片识别智能化
- 边缘部署适配
