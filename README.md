# 绿萝监测系统 v3

> 办公室绿萝生长监测系统 - Java + Vue 重构版

## 技术栈

- **前端**: Vue 3 + Element Plus
- **后端**: Java (Spring Boot)
- **数据库**: SQLite
- **存储**: 本地文件系统 + 图片

## 项目状态

**当前版本**: v3.0.0 (规划中)

**开发阶段**: 基础设施搭建
